# 제약구매팀 경제지표 대시보드

매일 업데이트되는 환율, 유가, 원자재 가격을 한눈에 확인하는 대시보드입니다.

## 주요 기능

- 💰 주요 환율 (USD/KRW, EUR/KRW, JPY/KRW, CNY/KRW)
- 🤝 협상용 크로스 환율 (USD/JPY, USD/CNY)
- ⛽ 국제 유가 (WTI, Brent)
- 🏭 주요 원자재 (금, 구리)
- 📰 제약업계 경제 뉴스
- 📧 이메일 브리핑 미리보기

## 사용 방법

1. 🔄 새로고침 버튼으로 최신 데이터 업데이트
2. 📧 이메일 미리보기로 브리핑 형식 확인
3. 🤝 크로스 환율은 해외 도매상 가격 협상 시 활용
4. ⛽ 유가 상승 시 물류비용 재협상 검토

## 기술 스택

- React
- Vite
- Tailwind CSS
- Anthropic Claude API
